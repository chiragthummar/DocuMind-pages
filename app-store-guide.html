Prerequisites & Apple Developer Account

Apple Developer Program

  • Enroll at developer.apple.com/programs ($99/year)
  • Wait for enrollment approval (usually 24-48 hours)
  • Sign in to App Store Connect

Xcode Configuration

  • Xcode 16+ installed
  • Signed in with your Apple Developer account in Xcode → Settings → Accounts
  • Bundle ID: com.shayona.documind
  • Team selected in Signing & Capabilities
  • Automatic signing enabled

Certificates & Profiles

With automatic signing, Xcode handles this. If manual:

  • Create iOS Distribution Certificate in Apple Developer portal
  • Create App Store provisioning profile for com.shayona.documind
  • Download and install both in Xcode

Xcode Archive & Upload

Pre-Archive Checklist

  • Set isPro = false in StoreKitService.swift (disable testing override)
  • Version number matches App Store Connect (e.g., 1.6)
  • Build number is unique and incremented
  • Scheme set to Release (not Debug)
  • Device set to "Any iOS Device (arm64)"
  • All signing certificates and profiles are valid
  • No compiler warnings (optional but recommended)

Disable Pro Testing Override

// In StoreKitService.swift, change: var isPro: Bool = true // ← TESTING // To: var isPro: Bool = false // ← PRODUCTION // Also in checkEntitlement(), change: isPro = true // ← TESTING // To: isPro = foundTier != nil // ← PRODUCTION

⚠️ Critical: Forgetting to disable the pro override means ALL users get Pro features for free!

Archive

  1. In Xcode: Product → Archive (or Cmd+Shift+B to build, then Product → Archive)
  2. Wait for archive to complete
  3. Organizer window opens automatically

Upload to App Store Connect

  1. In Organizer: Select the archive → Distribute App
  2. Choose App Store Connect
  3. Choose Upload
  4. Leave all options checked (Strip Swift Symbols, Upload Symbols, Manage Version)
  5. Select your Distribution Certificate and Profile
  6. Click Upload
  7. Wait for processing (5-15 minutes)

Command Line Alternative

# Archive xcodebuild archive \ -project DocuMind.xcodeproj \ -scheme DocuMind \ -archivePath ~/Desktop/DocuMind.xcarchive \ -destination "generic/platform=iOS" # Export for App Store xcodebuild -exportArchive \ -archivePath ~/Desktop/DocuMind.xcarchive \ -exportPath ~/Desktop/DocuMind-Export \ -exportOptionsPlist ExportOptions.plist # Upload via altool or Transporter app
